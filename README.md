# Inverted Search

## Overview
Inverted Search is a C programming project that creates an index of words from multiple text files and enables efficient searching of words along with the files in which they appear.

## Features
- Create an inverted index from multiple text files
- Search for a word efficiently
- Display the files containing the searched word
- Update the database with new files
- Save and load the database
- Dynamic memory allocation
- File handling

## Technologies Used
- C Programming
- Data Structures
- Linked Lists
- Hash Tables
- File Handling

## Project Files
- main.c
- create_database.c
- update_database.c
- search.c
- display_database.c
- save_database.c
- read_and_validate.c
- *.h (Header Files)

## How to Run
1. Compile the project using GCC.
2. Execute the program.
3. Provide the required text files as input.
4. Choose the desired operation from the menu.
